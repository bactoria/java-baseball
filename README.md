# Java Baseball

> 우아한 테크 코스 :: 온라인 과제 1주차
   
&nbsp;

## 기능 목록

- Baseball
    + Player 숫자 입력받기
    + 정답 여부에 따른 처리
        * 정답 : 게임종료
        * 오답 : 힌트 출력 후, 숫자 다시 입력받기
    + 게임 재시작 여부에 따른 처리
        * 1 : 게임 재시작
        * 2 : 프로세스 종료
- Computer
    + 랜덤 3자리 정수 생성
    + 정답 여부를 알려주는 기능
    + 힌트를 알려주는 기능
- Player
    + 숫자 setter 기능
    + 숫자 getter 기능
- Hint
    + 힌트를 계산하는 기능
